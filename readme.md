## Run

- Run `./gradlew build` to build the project.
- Run `./gradlew run` to start the game.

## Commands

- `d x y` - Dig at coordinates (x, y)
- `f x y` - Toggle flag at coordinates (x, y)
- `g x y` - Guess (dig based on flags) at coordinates (x, y)
